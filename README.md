# Видеоигра Arkanoid

Перемещайте платформу клавишами "вправо-влево" и разбивайте кирпичи ударом отскочившего от нее мячика.  
Ваша цель - разбить все кирпичи и не дать мячику улететь вниз за пределы игрового поля.
